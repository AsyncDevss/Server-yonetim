# 🛠️ Discord Advanced Tools (Çok Amaçlı Araç Seti)

Discord sunucu sahipleri ve yöneticileri için günlük işleri kolaylaştıran, tek dosya üzerinden çalışan işlevsel bir yönetim ve analiz araç setidir.

## ✨ Komutlar ve Özellikler
* `.temizle [miktar]` -> Kanaldaki belirtilen sayıda mesajı tek seferde uçurur.
* `.roldagit [@Rol]` -> Sunucudaki bot hariç tüm üyelere tek tıkla rol tanımlar.
* `.rolsuzver [@Rol]` -> Sunucuya girip hiçbir rol alamamış/kaydolmamış kişileri bulur ve toplu rol verir.
* `.davetler` -> Sunucuda kimin davetiyle kaç kişi gelmiş, aktif linklerle beraber listeler.
* `.sunucubilgi` -> Sunucunun üye, kanal, takviye ve teknik detaylarını şık bir panelle gösterir.
